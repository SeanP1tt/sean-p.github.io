In order to give you practice constructing websites, I want you to try to make a page similar to this, but with your own take on it. You can make a webpage about yourself, a business idea you've had, or even something you have interest in (like cats).

The possibilities are endless. (I don't like cats that much, this was just the quickest example).

Be Sure to Inspect This Webpage to See How it was Built


All you need to do is add 5 unique tags to an html page excluding the body, head, and html tags. You can add as much text as you want (It will make the page look better if you add more, but proper spacing is also important). Try to divide the text amongst different paragraph tags or even using headers between differen topics amongst the page.

Html is the easy part. So when you're done try adding a css page and styling some of the elements. I'll provide an example stylesheet as well for you to add to and tweak.
